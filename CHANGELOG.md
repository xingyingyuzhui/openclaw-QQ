# Changelog

## v0.1.1 - 2026-02-27
- Added detailed NapCat deployment guide for Docker and non-Docker setups.
- Added explicit version baseline and compatibility matrix.
- Added OneBot v11 required config (`messagePostFormat=array`, WS server/token mapping).
- Expanded AGENTS install/verify/troubleshooting instructions.

- Initial open-source release.
- Includes `qq` channel plugin and `qq-automation-manager` plugin.
- Route-isolated sessions, media inbound/outbound pipeline, and structured diagnostics.
- Automation manager runs in `agent-only` mode by default.
